# Blackout
> What do you do when the lights go out?

*Made with love and zero sleep by the **PERL** team*


# { TLDR }
Blackout is a CO-OP timed minigame that will push you and your friends ability to cooperate, communicate, and ultimately escape the backrooms.

Features 5 iconic floors from the backrooms, 3 new monsters, and dozens of new and interesting items and mechanics. 

You will quickly find, however, that this is not just your typical backrooms minigame...


# { Getting Started } 
1. Download the .zip file holding both Gridwave (a dependency) and Blackout (this mod)
2. Put both into your mods/ folder
3. Create a new world
4. Invite a few friends (not required)
5. Enter the backrooms! (you have to noclip through one of the walls... ||hint: look for the red arrow||

Once you are in, it is up to you to explore and discover this new, strange world.


# { GOOD TO KNOW }
- First load may take awhile as it dynamically generates a map for you (unique every time)
- Do `/blackout reset` to reset your timer and progress (do not abuse - this is a debug command)
- You may die the first time you no-clip. What were you expecting? It's not exactly safe
- Try NOT using your map for a greater challenge :P
- All music is made by Ev0 - it is free to use in any content/videos


# { Troubleshooting }
- Ensure there are no other mods installed
- More will be added here as people encounter problems
- This is a MODIFIED version of Gridwave - it must be the one included in the zip file
 


# { Spoilers }
The following have hints from generic to specific. ONLY REVEAL AS YOU GET STUCK. 

## Room #1: Office Rooms
1. Try breaking everything
2. Run for your life!
3. Avoid setting up base until you have ample resources
4. Use your flashlight to scare off the seeker
5. Craft daggers and kill the dummies before there are too many
6. Setup fortifications around your Corruption Machine - it will take a few nights to corrupt the key!
7. Find the ladder, it will lead you to the corrupted door

## Room #2: Pool Rooms
1. Avoid the water during the day - it boils
2. Try following the red arrows...
3. One arrow points down

## Room #3: Garage Floor
1. When the alarms go off, go into a sewage drain until it passes. DO NOT OPEN THE HATCH. It will get you
2. Try collecting some resources and looking for something out of place
3. Find the crafting bench, collect the resources needed
4. Find a garage door, interact with the garage door using the remote

## Room #4: Infinite Staircase
1. just keep going up - you got this

## Room #5: Level 10
1. this pillars look weird...
2. go to a pillar
